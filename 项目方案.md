# 项目方案

## 项目一：基于VLM的智能驾驶场景结构化理解系统

### 1. 项目定位

输入一张驾驶场景图片，系统自动输出结构化的场景描述（车道线、车辆、交通标志、危险判断），替代人工逐帧标注。

### 2. 分阶段实施路径

#### 阶段一：Baseline搭建

1. 选模型：Qwen2.5-VL-3B-Instruct
2. 搭建推理 pipeline：加载模型权重 → 写 prompt 模板 → 跑通单张推理
3. 搭建评估框架：50张测试图，定义四维度评估标准，人工打分

#### 阶段二：数据准备与微调

1. 数据集构建：nuScenes 标注 → VLM 对话格式
2. LoRA 微调：PEFT + QLoRA（4bit量化），rank=8, alpha=16
3. 对比实验：微调前 vs 微调后，记录 bad case

#### 阶段三：Demo封装与展示

1. Gradio 搭建前端
2. 模型推理封装为 DrivingSceneAnalyzer 类
3. README + 效果图

### 3. 技术栈

| 组件 | 选择 | 备注 |
|------|------|------|
| 基座模型 | Qwen2.5-VL-3B-Instruct | 中文友好，3B参数 |
| 微调框架 | PEFT + QLoRA | 显存友好，单卡可跑 |
| 数据集 | nuScenes | 自动驾驶标准数据集 |
| 前端Demo | Gradio | 快速出效果 |
