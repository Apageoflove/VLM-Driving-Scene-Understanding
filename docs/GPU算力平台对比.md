# GPU算力平台对比

> 用于 VLM LoRA 微调场景（32GB显存），单卡训练约15分钟。

## 推荐排序

| 优先级 | 平台 | 4090价格 | 特点 | 注册 |
|------|------|----------|------|------|
| 1 | **恒源云** | 1.32-1.8元/时 | 老牌稳定、用户少易抢到、免费存储空间 | 直接注册 |
| 2 | **AutoDL** | 1.8-2.8元/时 | 资源最丰富、社区最活跃、按秒计费 | 直接注册 |
| 3 | 晨涧云 | 1.5-2.5元/时 | 性价比高、支持Windows | 收不到验证码，已放弃 |

## 不推荐

| 平台 | 原因 |
|------|------|
| 阿里云 | GPU新号开不了实例，按量15-28元/时太贵 |
| 华为云 | 昇腾芯片，不兼容CUDA生态，Qwen2.5-VL+LoRA方案要重写 |

## 策略

恒源云主力 + AutoDL备用，哪家有卡跑哪家。训练脚本只需改 `base` 路径。

## 华为云不适用原因

- 用的是昇腾（Ascend）芯片，不是NVIDIA GPU
- bitsandbytes、flash-attention 等关键库不兼容
- QLoRA 量化微调在昇腾上没有现成方案，迁移成本远高于算力费差价
